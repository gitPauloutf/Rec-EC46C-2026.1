# 1. Requisitos Funcionais

<p align="justify">A <i>Tabela 1</i> a seguir contém os Requisitos Funcionais (RF).</p>

| ID   | Requisito | Prioridade | Requisitos Relacionados |
| :--: | :-----------------------------------------------------------------------------------------------------------------------------------: | :--------: | :----------------------: |
| RF01 | O sistema deve permitir que estudantes realizem cadastro, login e gerenciamento de seu perfil para acessar a plataforma. | Alta | - |
| RF02 | O sistema deve permitir criar, editar, visualizar, abandonar e dissolver grupos de estudo, sendo que apenas o organizador poderá editar ou dissolver um grupo. | Alta | RF01 |
| RF03 | O sistema deve permitir que o organizador adicione e remova estudantes dos grupos de estudo sob sua responsabilidade. | Alta | RF01, RF02 |
| RF04 | O sistema deve permitir que o organizador crie, edite e visualize reuniões dos grupos de estudo, informando data, horário e descrição. | Alta | RF02 |
| RF05 | O sistema deve disponibilizar um quadro de discussões para que os participantes de um grupo possam trocar mensagens e discutir assuntos relacionados aos estudos. | Média | RF02 |
| RF06 | O sistema deve permitir que os participantes compartilhem arquivos e materiais de estudo dentro dos grupos dos quais fazem parte. | Média | RF02 |
