d. O projeto busca auxiliar a formação e a organização de grupos de estudo para alunos. Nele, usuarios podem organizar grupos (se tornando organizadores deste) e convidar alunos. O organizador pode então marcar reuniões e adicionar outros alunos, enquanto todos usuarios podem compartilhar arquivos ou conversar na plataforma.
