▪ O que é Integração Contínua?  
É a prática de integrar alterações de código com frequência em um repositório compartilhado, usando testes automáticos para encontrar erros rapidamente a cada integração.  

▪ Qual é o papel das branches nesse processo?  
Permitem desenvolver funcionalidades ou corrigir erros de forma isolada, sem afetar o código principal. Quando o trabalho é finalizado e verificado, a branch é integrada a branch principal (main).  

▪ Por que o Pull Request é importante?  
Porque permite revisar e testar o código antes de juntá-lo à branch principal, garantindo estabilidade do projeto.
